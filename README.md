# Sales Insights Dashboard (Tableau)

## Project Overview

This project presents an interactive dashboard built using Tableau to analyze sales performance.

## Tools Used

* Tableau
* CSV Dataset (Superstore)

## Key Features

* Sales and Profit Analysis
* Region-wise and Category-wise insights
* Time-series trend analysis
* Interactive filters

## Insights

* Identified top-performing regions
* Analyzed profit variations across categories

## How to Use

Download the .twbx file and open in Tableau to explore the dashboard.
